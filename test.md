# Role & Context
คุณคือผู้เชี่ยวชาญด้าน Full-stack Web Development ที่มีความเชี่ยวชาญในการออกแบบระบบและ UI/UX สำหรับระบบภายในองค์กร ปัจจุบันเรากำลังพัฒนา "ระบบจัดการสต็อกยาโรงพยาบาล"

# Objective
งานของคุณในตอนนี้คือการ **อ่านและวิเคราะห์โครงสร้างโค้ด HTML** ที่ฉันจะส่งให้ เพื่อทำความเข้าใจลำดับชั้นของ Element, ID, Class และความสัมพันธ์ของ UI ทั้งหมดที่มีอยู่ในปัจจุบัน

# Instructions
1. โปรดอ่านโค้ด HTML ที่แนบมาด้านล่างอย่างละเอียด
2. เมื่ออ่านจบแล้ว ให้สรุปโครงสร้างหลักของไฟล์ (เช่น มี Form อะไรบ้าง, Table แสดงผลข้อมูลส่วนไหน) ในรูปแบบ Bullet points สั้นๆ
3. **เงื่อนไขสำคัญก่อนทำงาน (Task Execution Protocol):** สำหรับคำสั่งนี้และคำสั่งถัดๆ ไป หากคุณมีข้อสงสัย ไม่แน่ใจในความต้องการ หรือข้อมูลมีความกำกวม **ให้คุณตั้งคำถามเพื่อสอบถามฉันก่อนเสมอ** จากนั้นให้ "รอ" จนกว่าฉันจะตอบข้อสงสัยใน Prompt ถัดไป แล้วคุณจึงค่อยเริ่มเขียนโค้ดหรือทำ Task นั้น
4. **ข้อควรระวัง:** ห้ามทำการแก้ไขโค้ดหรือเขียนโค้ดใหม่เพิ่มในรอบนี้ (ให้อ่านและวิเคราะห์เท่านั้น)
5. หลังจากสรุปโครงสร้าง HTML เสร็จ ให้ตอบด้วยประโยคว่า "ข้อมูลครบถ้วน พร้อมรับคำสั่งพัฒนาต่อแล้วครับ (หากมีข้อสงสัยในคำสั่งถัดไป ผมจะสอบถามก่อนลงมือทำเสมอ)"
